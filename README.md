Data Science Portfolio: Machine Learning & Analysis Tasks
This repository contains a collection of data science and machine learning projects focusing on classification, regression, and exploratory data analysis (EDA). The projects utilize popular Python libraries including Pandas, Scikit-Learn, Matplotlib, and Seaborn.

Project Breakdown:

Task 1: Iris Flower Classification
Objective: Classify Iris flowers into three distinct species (Setosa, Versicolor, Virginica) based on physical measurements (sepal and petal length/width).
Type: Multi-class Classification (Supervised Learning).
Key Concepts:Feature scaling, classification metrics, decision boundaries.

Task 2: Unemployment Analysis with Python
Objective:Analyze economic datasets to uncover sharp shifts, seasonal trends, and regional impact patterns in the unemployment rate.
Type:Exploratory Data Analysis (EDA) & Data Visualization.
Key Concepts:Time-series line plots, regional breakdown charts, pre/post Covid-19 economic impact analysis.

Task 3: Car Price Prediction with Machine Learning
Objective:Estimate the used vehicle marketplace value using predictive factors like historical wear, retail value, and operational mileage.
Type:Regression (Supervised Learning).
Key Concepts:Handling data typos, category conversion (One-Hot Encoding), feature age calculation.

Task 4: Sales Prediction using Python
Objective:Forecast product market demand based on budget allocations across distinct advertising platforms (TV, Radio, Newspaper).
Type:Regression & Feature Selection.
Key Concepts:Linear regression coefficients, Return on Investment (ROI) calculation, correlation matrix heatmaps.
